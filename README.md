# SecondForum
A second forum for who lost their home.

## 前言

這是想要做出一個跟PTT體驗相似的純文字論壇，目前的目標是

1.快速，0延遲的操作
2.可以全鍵盤操作/可以全滑鼠操作

目前前端打算用React，後端打算用golang，資料庫還沒決定

因為大部分的框架及語言都是初學，所以應該會弄很久
